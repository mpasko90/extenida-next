# Extenida Next

## Overview

Modern, accessible, and SEO-optimized bathroom renovation website built with Vite and React.

### Quick Start

1. Install dependencies:

```bash
npm install
```

1. Start development server:

```bash
npm run dev
```

1. Build for production:

```bash
npm run build
```

1. Preview production build:

```bash
npm run preview
```
